# Detailed Feature Specifications

> This directory contains detailed specifications for project features.
> Use these files as "Executable Context" for AI agents.

## Structure

- Create a Markdown file for each major feature or complex task.
- Use the `{task-slug}.md` naming convention.
- Reference these files in chat using their path or unique identifiers.

## Examples

- `auth-flow.md`
- `payment-integration.md`
- `dashboard-ui.md`
